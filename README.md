My exploration scripts for virtual rodent imtiation.

## Dependecies
Uses [vnl-playground](https://github.com/talmolab/vnl-playground) for environment specfication and [NNX-PPO](https://github.com/emiwar/nnx-ppo/) to train the networks.

## See also
[track-mjx](https://github.com/talmolab/track-mjx) Training vnl-playground environments specifically with encoder-decoder architectures
[julia-virtual-rodent](https://github.com/emiwar/julia-virtual-rodent) Rodent imitation training, including modular networks, with both network and environment in Julia
